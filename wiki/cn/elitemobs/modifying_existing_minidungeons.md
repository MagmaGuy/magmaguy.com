# 修改现有小型地下城

根据大家的要求，本页面将介绍修改现有小型地下城的基础知识。这不是一个分步指南，而是更广泛地介绍如何修改这些系统。您需要阅读关于[自定义首领]($language$/elitemobs/creating_bosses.md)和[自定义物品]($language$/elitemobs/creating_items.md)的维基页面，以便修改这些值。

### 修改地下城难度

您可能希望修改地下城难度有三个原因，具体如下：

### 修改首领的伤害/生命值乘数

这是最简单的修改。如果您认为首领造成的伤害过低或过高，或者他们的生命值过低或过高，您可以修改`custombosses`文件夹中首领文件中与这些乘数相关的值。

您可能希望修改的值示例：
```yml
healthMultiplier: 0.5
damageMultiplier: 0.5
```
更多关于首领属性的信息请见[此处]($language$/elitemobs/creating_bosses.md&section=healthmultiplier)。

请记住，对于许多能力来说，伤害乘数不会修改能力造成的伤害。请参阅下一节了解如何修改能力。

### 修改首领的能力

EliteMobs 的主要难度在于处理首领可能拥有的各种能力。等级较高的首领通常拥有更多更强的能力，而等级较低的首领则拥有更少更弱的能力。某些能力在同一个首领身上一起使用会使战斗更容易，而其他能力组合在一起则会使战斗显著更难。

您可能希望修改的值示例：
```yml
powers:
- invulnerability_fire.yml
- invulnerability_arrow.yml
```
更多关于首领能力的信息请见[此处]($language$/elitemobs/creating_bosses.md&section=powers)。

由于能力组合太多无法在此列出，您需要学习这些能力的工作原理并亲自测试您的遭遇战，以便找到您认为既不太难也不太容易的遭遇战的平衡点。

### 修改首领的等级

这是最常见的问题，虽然可以做到，但这需要几个步骤来确保您适当地提升或降低首领的等级。

您可能希望修改的值示例：
```yml
level: 21
```
更多关于首领等级的信息请见[此处]($language$/elitemobs/creating_bosses.md&section=level)。

这需要结合前面提到的伤害/生命值乘数，因为较低/较高等级的首领可能需要略微不同的生命值和伤害乘数才能使战斗感觉合适。

然而，这里最重要的考虑因素是首领的能力。较低等级和较高等级的首领通常根据战斗的预期难度拥有相当不同的能力组合。对于一个10级首领和一个200级首领使用相同的能力组合，通常会导致战斗难度感觉截然不同。同样，找到真正平衡点的唯一方法是学习插件能力的工作原理并亲自测试它们。

最后，**但同样重要！** 您可能需要调整自定义掉落物。EliteMobs 中的大多数首领都有自定义掉落物，如果您更改首领的等级，很可能您会希望改变其掉落物的性质。

您可能希望修改的值示例：
```yml
uniqueLootList:
  - filename: mob_drop_boots.yml
    chance: 0.05
```
更多关于掉落物的信息请见[此处]($language$/elitemobs/loot_tables.md)。

### 署名 / 发布修改后的内容

在 EliteMobs 下，您可以自由地在适当的渠道（Discord）发布您创建的地下城。但是，如果您的创作只是对现有地下城的修改，您的提交可能会被移除，特别是如果它只是一个小的调整，或者如果它包含或源自付费内容（出于显而易见的原因）。

您可以自由并欢迎修改您购买或以其他方式下载的任何内容，以根据您服务器的需求在您自己的网络中使用，但出于显而易见的原因，这不赋予您再分发权。

我们保留出于任何原因随时删除社区提交内容的权利，尽管我们非常不愿意这样做。
