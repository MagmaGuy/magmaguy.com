# 预设能力列表

一个Boss可以拥有一个或几个能力。我们建议你将数量保持在5以下。完整的能力列表可以在`~/plugins/EliteMobs/powers`文件夹中找到。

以下列表并不一定完整，因为[你可以创建自定义能力]($language$elitemobs/creating_powers.md)，并且一些地牢带有它们自己的能力。

*注意：点击[这里]($language$/elitemobs/creating_bosses.md&section=powers)学习如何使用这些能力自定义Boss。*

能力的GIF仅供演示之用。请注意其动画并**不** 反映游戏中能力的实际动画速度。

# 精英脚本格式

请注意，有些能力有[精英脚本]($language$/elitemobs/creating_powers.md)格式，并可以自定义。

以下是*attack_blinding.yml*能力的一个例子，说明了如何使用[精英脚本]($language$/elitemobs/creating_powers.md)格式进行配置：

<div align="left">

```yml
isEnabled: true
effect: SPELL_MOB
powerType: MISCELLANEOUS
eliteScript:
  BlindPlayer:
    Events:
      - PlayerDamagedByEliteMobEvent
    Actions:
      - potionEffectType: blindness
        amplifier: 0
        action: POTION_EFFECT
        duration: 60
        Target:
          targetType: DIRECT_TARGET
```

这个特性使您可以完全自定义*attack_blinding.yml*能力。您可以指定应该应用哪种药水效果，效果的持续时间，目标和触发boss使用能力的事件。

</div>
