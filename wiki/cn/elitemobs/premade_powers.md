# 预制力量列表

Boss 可以拥有一种或多种力量。我们建议您将数量保持在 5 以下。力量的完整列表可以在 `~/plugins/EliteMobs/powers` 文件夹中找到。

以下列表不一定完整，[因为您可以制作自定义力量]($language$elitemobs/creating_powers.md)，并且某些地下城带有自己的力量。

*注意：点击 [此处]($language$/elitemobs/creating_bosses.md&section=powers) 了解如何使用这些力量自定义 Boss。*

力量 GIF 仅用于演示目的。请注意，它们的动画**不**反映游戏中的实际力量动画速度。

# Elite 脚本格式

请注意，某些力量采用 [Elite 脚本]($language$/elitemobs/creating_powers.md) 格式，并且可以自定义。

以下是一个 *attack_blinding.yml* 力量的示例，以及如何使用 [Elite 脚本]($language$/elitemobs/creating_powers.md) 格式对其进行格式化：

<div align="left">

```yml
isEnabled: true
effect: SPELL_MOB
powerType: MISCELLANEOUS
eliteScript:
  致盲玩家:
    事件:
      - PlayerDamagedByEliteMobEvent
    动作:
      - potionEffectType: 失明
        amplifier: 0
        action: 药水效果
        duration: 60
        目标:
          目标类型: 直接目标
```

此功能使您能够完全自定义 *attack_blinding
