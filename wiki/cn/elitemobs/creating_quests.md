# 示例任务

EliteMobs 预装了一个 `test_quest.yml` 文件，这里将对其进行分析，作为一个简单的任务格式供您参考。

_自定义任务位于 `~plugins/EliteMobs/customquests` 文件夹中！_

`test_quest.yml`

```yaml
isEnabled: true
customObjectives:
  Objective1:
    amount: '1'
    filename: test_boss.yml
    objectiveType: KILL_CUSTOM
customRewards:
- filename=magmaguys_toothpick.yml:amount=1:chance=1
name: "&a击杀测试 Boss"
questLore:
- "&c终结测试 Boss 的恐怖统治！"
```

<div align="center">

![create_quest_quest.jpg](../../../img/wiki/create_quest_quest.jpg)

</div>

*如果您的任务 UI 不像这样，您可以使用 `/em alt` 命令更改它以匹配。*

这个示例任务将要求玩家击杀 1 个 test_boss.yml。（在任务追踪器中显示的 Boss 实际名称将是 test_boss.yml 中设置的 `name:`。）完成任务后，他们将获得 1 个 Magmaguy's Toothpick 作为奖励。

## 创建自定义任务

<div align="center">

### customObjectives

设置任务目标。

| Key              |   Values    | Default |
|------------------|:-----------:|:-------:|
| `customObjectives` | 特殊 [1] |  无   |

*注意：如果您使用多阶段 Boss 作为您的目标，则目标应使用第一阶段作为目标。*

<details>

<summary><b>示例</b></summary>

<div align="left">

`KILL_CUSTOM`:

```yml
customObjectives:
  Objective1:
    amount: '1'
    filename: my_cool_boss.yml
    objectiveType: KILL_CUSTOM
```

`DIALOG`:

```yml
customObjectives:
  Objective1:
    dialog:
    - "&a[对话 NPC] &f常来这里吗？"
    - "&7&o我应该多吃点苹果。"
    filename: dialog_npc.yml
    npcName: Dialog NPC
    location: at dialog location.
    objectiveType: DIALOG
```

`FETCH_ITEM`:

```yml
customObjectives:
  Objective1:
    amount: '99'
    itemName: Red Apples
    filename: my_quest_item_red_apples.yml
    objectiveType: FETCH_ITEM
```

# Example Quest

EliteMobs comes prepackaged with a `test_quest.yml`, which will be analyzed here as a simple quest format to follow.

_Custom Quests are in the `~plugins/EliteMobs/customquests` folder!_

`test_quest.yml`

```yaml
isEnabled: true
customObjectives:
  Objective1:
    amount: '1'
    filename: test_boss.yml
    objectiveType: KILL_CUSTOM
customRewards:
- filename=magmaguys_toothpick.yml:amount=1:chance=1
name: "&a击杀测试 Boss"
questLore:
- "&c终结测试 Boss 的恐怖统治！"
```

<div align="center">

![create_quest_quest.jpg](../../../img/wiki/create_quest_quest.jpg)

</div>

*如果您的任务 UI 不像这样，您可以使用 `/em alt` 命令更改它以匹配。*

这个示例任务将要求玩家击杀 1 个 test_boss.yml。（在任务追踪器中显示的 Boss 实际名称将是 test_boss.yml 中设置的 `name:`。）完成任务后，他们将获得 1 个 Magmaguy's Toothpick 作为奖励。

## 创建自定义任务

<div align="center">

### customObjectives

设置任务目标。

| Key              |   Values    | Default |
|------------------|:-----------:|:-------:|
| `customObjectives` | 特殊 [1] |  无   |

*注意：如果您使用多阶段 Boss 作为您的目标，则目标应使用第一阶段作为目标。*

    objectiveType: KILL_CUSTOM
```

`DIALOG`:

```yml
customObjectives:
  Objective1:
    dialog:
    - "&a[对话 NPC] &f常来这里吗？"
    - "&7&o我应该多吃点苹果。"
    filename: dialog_npc.yml
    npcName: Dialog NPC
    location: at dialog location.
    objectiveType: DIALOG
```

`FETCH_ITEM`:

```yml
customObjectives:
  Objective1:
    amount: '99'
    itemName: Red Apples
    filename: my_quest_item_red_apples.yml
    objectiveType: FETCH_ITEM

## Creating Custom Quests

<div align="center">

### customObjectives

设置任务目标。

| Key              |   Values    | Default |
|------------------|:-----------:|:-------:|
| `customObjectives` | 特殊 [1] |  无   |

*注意：如果您使用多阶段 Boss 作为您的目标，则目标应使用第一阶段作为目标。*

<details>

<summary><b>示例</b></summary>

<div align="left">

`KILL_CUSTOM`:

```yml
customObjectives:
  Objective1:
    amount: '1'
    filename: my_cool_boss.yml
    objectiveType: KILL_CUSTOM
```

`DIALOG`:

```yml
customObjectives:
  Objective1:
    dialog:
    - "&a[对话 NPC] &f常来这里吗？"
    - "&7&o我应该多吃点苹果。"
    filename: dialog_npc.yml
    npcName: Dialog NPC
    location: at dialog location.
    objectiveType: DIALOG
```

`FETCH_ITEM`:

```yml
customObjectives:
  Objective1:
    amount: '99'
    itemName: Red Apples
    filename: my_quest_item_red_apples.yml
    objectiveType: FETCH_ITEM
```

# 示例任务

EliteMobs 预装了一个 `test_quest.yml` 文件，这里将对其进行分析，作为一个简单的任务格式供您参考。

_自定义任务位于 `~plugins/EliteMobs/customquests` 文件夹中！_

`test_quest.yml`

```yaml
isEnabled: true
customObjectives:
  Objective1:
    amount: '1'
    filename: test_boss.yml
    objectiveType: KILL_CUSTOM
customRewards:
- filename=magmaguys_toothpick.yml:amount=1:chance=1
name: "&a击杀测试 Boss"
questLore:
- "&c终结测试 Boss 的恐怖统治！"
```

<div align="center">

![create_quest_quest.jpg](../../../img/wiki/create_quest_quest.jpg)

</div>

*如果您的任务 UI 不像这样，您可以使用 `/em alt` 命令更改它以匹配。*

这个示例任务将要求玩家击杀 1 个 test_boss.yml。（在任务追踪器中显示的 Boss 实际名称将是 test_boss.yml 中设置的 `name:`。）完成任务后，他们将获得 1 个 Magmaguy's Toothpick 作为奖励。

## 创建自定义任务

<div align="center">

### customObjectives

设置任务目标。

| Key              |   Values    | Default |
|------------------|:-----------:|:-------:|
| `customObjectives` | 特殊 [1] |  无   |

*注意：如果您使用多阶段 Boss 作为您的目标，则目标应使用第一阶段作为目标。*

    objectiveType: KILL_CUSTOM
