# 既存のミニダンジョンの変更

多くのリクエストに基づき、このページでは、既存のミニダンジョンの変更の基本について説明します。これは、手順を追ったガイドではなく、システムを変更する方法のより広範な概要です。これらの値を変更するには、[カスタムボス]($language$/elitemobs/creating_bosses.md) と [カスタムアイテム]($language$/elitemobs/creating_items.md) の wiki ページを読む必要があります。

### ダンジョンの難易度を変更する

ダンジョンの難易度を変更する必要がある理由は 3 つあります。

### ボスのダメージ/体力倍率を変更する

これは、最も簡単な変更です。ボスが与えるダメージが少なすぎる、または多すぎる、または体力があまりにも少ない、または多すぎると思う場合は、`custombosses` フォルダにあるボスファイルに含まれるこれらの倍率に関連する値を変更できます。

変更する必要がある値の例を以下に示します。
```yml
healthMultiplier: 0.5
damageMultiplier: 0.5
```
ボスのステータスについては、[こちら]($language$/elitemobs/creating_bosses.md&section=healthmultiplier) をご覧ください。

多く のパワーでは、ダメージ倍率がパワーによるダメージに影響を与えないことに注意してください。パワーの変更方法については、次のセクションを参照してください。

### ボスのパワーを変更する

EliteMobs の主な難しさは、ボスが持つさまざまなパワーを扱うことです。レベルの高いボスは、より多く、より強力なパワーを持つ傾向があり、レベルの低いボスは、パワーが少なく、容易な傾向があります。同じボスに特定のパワーを組み合わせると、戦闘が容易になる場合もあれば、大幅に難しくなる場合もあります。

変更する必要がある値の例を以下に示します。
```yml
powers:
- invulnerability_fire.yml
- invulnerability_arrow.yml
```
ボスパワーについては、[こちら]($language$/elitemobs/creating_bosses.md&section=powers) をご覧ください。

ここでは、リストできるパワーの組み合わせが多すぎるため、パワーの仕組みを理解し、遭遇をプレイテストして、難しすぎず、簡単すぎない遭遇を見つける必要があります。

### ボスのレベルを変更する

これは、最も頻繁に尋ねられる質問であり、可能です。しかし、ボスを適切にレベルアップまたはレベルダウンするには、いくつかの手順が必要です。

変更する必要がある値の例を以下に示します。
```yml
level: 21
```
ボスのレベルについては、[こちら]($language$/elitemobs/creating_bosses.md&section=level) をご覧ください。

これは、ダメージ/体力倍率に関する説明と組み合わせたものです。レベルが低い/高いボスでは、戦闘を適切に感じさせるために、わずかに異なる体力とダメージ倍率が必要になる場合があります。

ただし、ここで最も重要なのは、ボスのパワーです。レベルの低いボスとレベルの高いボスでは、戦闘の難易度によって、パワーセットがかなり異なります。レベル 10 のボスとレベル 200 のボスに同じパワーセットを設定すると、難易度の感覚が大きく異なる戦闘になることがよくあります。繰り返しになりますが、適切なバランスを見つけるには、プラグインのパワーの仕組みを理解し、プレイテストを行うのが最善です。

最後に、**非常に重要なことですが！** カスタムlootを調整する必要があるかもしれません。EliteMobs のほとんどのボスは、カスタムlootを持っています。ボスのレベルを変更する場合は、ドロップする loot の内容を変更したくなるでしょう。

変更する必要がある値の例を以下に示します。
```yml
uniqueLootList:
  - filename: mob_drop_boots.yml
    chance: 0.05
```
loot ドロップについては、[こちら]($language$/elitemobs/loot_tables.md) をご覧ください。

### 変更したコンテンツのクレジット/公開

EliteMobs では、作成したダンジョンを適切なチャンネル（Discord）に投稿できます。ただし、作成したコンテンツが既存のダンジョンの修正に過ぎない場合は、特に小さな調整や、プレミアムコンテンツを含む/プレミアムコンテンツから派生したコンテンツの場合は、投稿が削除される可能性があります。

購入した、またはその他の方法でダウンロードしたコンテンツを変更して、自分のネットワーク内での使用に調整することは自由です。ただし、当然のことながら、再配布権は付与されません。

コミュニティの投稿は、いかなる理由であっても、いつでも取り下げる権利を留保します。ただし、実際にそのような措置をとることは避けたいと考えています。
