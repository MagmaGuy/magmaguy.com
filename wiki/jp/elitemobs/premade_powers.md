# プリメイドパワーリスト

ボスは1つまたは複数のパワーを持つことができます。5つ以下に抑えることをお勧めします。パワーの完全なリストは、`~/plugins/EliteMobs/powers`
フォルダーにあります。

以下のリストは、カスタムパワーを作成できるため必ずしも完全ではありません[カスタムパワーを作成できる](($language$elitemobs/creating_powers.md))
し、一部のダンジョンには独自のパワーが付属しています。

*注：これらのパワーでボスをカスタマイズする方法については、[こちら]($language$/elitemobs/creating_bosses.md&section=powers)
をクリックしてください。*

パワーのGIFは、デモンストレーションのみを目的としています。アニメーションは、ゲーム内の実際のパワーのアニメーション速度を*
*反映していない**ことに注意してください。

# エリートスクリプト形式

一部のパワーは[エリートスクリプト]($language$/elitemobs/creating_powers.md)形式であり、カスタマイズできることに注意してください。

以下は、*attack_blinding.yml*パワーの例と、[エリートスクリプト]($language$/elitemobs/creating_powers.md)形式を使用したフォーマット方法です。

<div align="left">

```yml
isEnabled: true
effect: SPELL_MOB
powerType: MISCELLANEOUS
eliteScript:
  BlindPlayer:
    Events:
      - PlayerDamagedByEliteMobEvent
    Actions:
      - potionEffectType: blindness
        amplifier: 0
        action: POTION_EFFECT
        duration: 60
        Target:
          targetType: DIRECT_TARGET
```

この機能を使用すると、*attack_blinding.yml*パワーを完全にカスタマイズできます。適用するポーション効果、効果の持続時間、ターゲット、ボスがパワーを使用するトリガーとなるイベントを指定できます。

</div>
