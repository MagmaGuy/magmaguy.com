# Максимальный уровень зачарования

Чтобы повысить максимальный уровень любого зачарования, перейдите в папку конфигурации `enchantments` EliteMobs и отредактируйте значение `maxLevelV2` желаемого зачарования.

# Критические удары

Критические удары дают шанс удвоить урон, наносимый элитному мобу.

Каждый уровень Критического удара увеличивает шанс критического удара на 10% (10% для уровня 1, 20% для уровня 2).

Предметы, сгенерированные процедурно, по умолчанию имеют максимальный уровень 2.

# Бурение

Зачарование Бурение — это зачарование для кирок. Оно позволяет киркам и лопатам, использующим его, добывать блоки в радиусе от добываемого блока. Чем выше уровень зачарования, тем больше радиус затрагиваемых блоков.

Уровни варьируются от 1 до 5: на уровне 1 добывается 1 дополнительный блок, а на уровне 5 — 9 дополнительных блоков. В этом радиусе будут добываться только блоки того же типа.

[Посмотрите это видео для наглядного представления того, как это работает.](https://youtu.be/CM78o_-Aa0s)

# Элитная Защита/Острота/Сила/Кара небесная/Небесная кара/...

Элитная Защита/Острота/Сила/Кара небесная/Небесная кара/... увеличивает уровень ванильного зачарования, но применяет это особое значение только при сражении с элитами.

**Предметы начинают получать Элитные зачарования только после превышения ванильных лимитов уровня зачарования (например, ванильная Защита имеет максимум 4 уровень). Все последующие уровни увеличиваются на 1, что означает, что предмет с Защитой 5 и Элитной Защитой 10 будет считаться броней с Защитой 15 при сражении с Элитными мобами, но будет считаться броней с Защитой 5 при сражении с любыми другими мобами.**

Эти зачарования применяются только при сражении с мобами из EliteMobs и не действуют в PvP или при сражении с другими мобами.

# Огнемет

Любой предмет с зачарованием Огнемет способен выпускать струю пламени, наносящую урон мобам в направлении выстрела.

При нажатии правой кнопки мыши для активации этого зачарования игрок становится неподвижным на время действия эффекта.

Использование зачарования Огнемет расходует прочность предмета.

Это пользовательское зачарование никогда не может наносить урон игрокам по соображениям баланса.

# Охотник

Зачарование Охотник увеличивает процент мобов вокруг игрока с этим зачарованием, которые превращаются в Элитных мобов, эффективно увеличивая количество Элитных мобов, появляющихся вокруг игрока.

Коэффициент превращения за каждый уровень настраивается, и каждый последующий уровень будет увеличивать этот коэффициент на указанную величину. Максимальный уровень зачарования Охотник — 3.

# Ледокол

Зачарование Ледокол позволяет любому предмету замораживать воду при ударе. Рекомендуется добавлять его только на кирки.

Зачарование Ледокол может иметь до двух уровней. На первом уровне кирка будет замораживать только ударенный блок. На втором — заморозит его и непосредственно прилегающие блоки.

Зачарование второго уровня можно использовать для создания туннелей под океаном.

[Посмотрите это видео, если хотите увидеть его в действии.](https://youtu.be/k206wfEBCqs)

# Землетрясение

Зачарование Землетрясение заставляет игрока совершать большой прыжок при двойном приседании в ботинках и отбрасывает мобов при приземлении игрока.

# Кошка

Зачарование Кошка, примененное к лукам, позволяет игрокам цепляться за [целевые блоки](https://minecraft.fandom.com/wiki/Target) при стрельбе любым типом стрел.

# Молния

Зачарование Молния работает аналогично зачарованию Критические удары, но наносит урон в 2.5 раза больше в радиусе 2.5 блока и вызывает удар молнии. Шанс срабатывания зачарования Молния очень низок.

# Громкие удары

Зачарование Громкие удары увеличивает агрессию при попадании по сущностям, созданным EliteMobs. Элиты и боссы обычно нацеливаются на игрока, нанесшего наибольший урон, но это зачарование позволяет танкам удерживать внимание моба, даже если они не наносят максимальный урон. Это используется для традиционных ролей танков в MMORPG. Это зачарование в основном используется для [Инстансированных подземелий]($language$/EliteMobs/building_for_elitemobs.md&section=instanced-dungeons) и находится в добыче, выпадающей из этих подземелий.

# Плазменные ботинки

Зачарование Плазменные ботинки при двойном приседании заставляет игрока совершать короткий прыжок, а затем создает зеленые частицы во всех направлениях, которые наносят урон сущностям при попадании. Это расходует прочность ботинок.

# Призыв торговца

Зачарование Призыв торговца призывает Дживса. Это делается нажатием правой кнопки мыши по предмету (обычно это просто бумага) или произнесением "Дживс!" в чате. Призыв Дживса расходует предмет с зачарованием.

# Призыв волка

Зачарование Призыв волка призывает дружелюбного волка при *Shift + нажатии правой кнопки мыши* по полу с предметом в руке. Уровень волка рассчитывается на основе снаряжения игрока, и он исчезает через 2 минуты. Его нельзя приручить.

Существует шанс 1/1000 призвать Снупи при призыве волка. Снупи намного сильнее обычных волков.

Использование зачарования Призыв волка расходует предмет, использованный для призыва.

# Отвязка

Зачарование Отвязка используется в свитке Отвязки как необходимое зачарование для меню / NPC отвязки. Подробнее см. в разделе [Привязка души](#Soulbind).

# Привязка души

Привязка души — это очень особое зачарование, и [с ним можно ознакомиться на этой странице]($language$/elitemobs/soulbind.md).
