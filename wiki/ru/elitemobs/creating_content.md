# Создание контента для EliteMobs

Создавать контент для EliteMobs легко, и это можно сделать всего за несколько минут!

На боковой панели вы найдете ссылки для создания всевозможного контента, от боссов до событий и целых подземелий!

## Перед началом

Вы можете создавать весь контент через [веб-приложение](https://magmaguy.com/webapp/webapp.html), что значительно ускоряет процесс.

Все файлы конфигурации используют формат yml/yaml. Это стандартный формат файлов конфигурации для Minecraft.

YML — это по сути текстовый файл, который следует определенным правилам форматирования для конкретных данных. Если вы планируете создавать много контента для EliteMobs и только начинаете, возможно, вам стоит ознакомиться с [этой страницей]($langage$/global/configuration_file_guide.md), которая показывает, как должны быть написаны текстовые файлы, чтобы иметь действительный формат yml.
