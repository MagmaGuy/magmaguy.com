# Пользовательская Таблица Добычи

Универсальный формат добычи. Используется EliteMobs в нескольких различных местах.

Обратите внимание, что это последний формат, который рекомендуется использовать, но существуют и некоторые альтернативные старые форматы.

Убедитесь, что ваши таблицы добычи начинаются с `uniqueLootList:` перед добавлением каких-либо настроек, иначе конфигурация .yml выдаст ошибку.

### Выпадение элитных предметов:

| Ключ | Подробности | Значения                | По умолчанию |
| --- | :-: |-----------------------| --- |
| `filename` | Устанавливает имя файла Пользовательского Предмета для использования. | [Имя файла](#filename) | нет |

<details>

<summary align="center"><b>Пример</b></summary>

<div align="left">

```yml
uniqueLootList:
  - filename: magmaguys_toothpick.yml
```

Это заставит моба выбросить 1 *Зубочистку МагмаГая* со 100% шансом выпадения.

</div>

</details>

### Общие настройки:

| Ключ | Подробности | Значения              | По умолчанию |
| --- | :-: |---------------------| --- |
| `chance` | Устанавливает шанс выпадения добычи. | [Двойное число](#double)   | `1.0` |
| `amount` | Устанавливает количество этого элемента добычи для выпадения | [Целое число](#integer) | `1` |
| `ignorePlayerLevel` | Заставляет выпадение добычи игнорировать ограничитель добычи игрока | [Булево](#boolean) | `false` |

<details>

<summary align="center"><b>Пример</b></summary>

<div align="left">

```yml
uniqueLootList:
  - filename: magmaguys_toothpick.yml
    chance: 0.5
    amount: 10
    ignorePlayerLevel: true
```

Это заставит моба выбросить 10 *Зубочисток МагмаГая* с 50% шансом выпадения, игнорируя уровень игрока.

</div>

</details>

### Выпадение элитных монет:

| Ключ | Подробности | Значения              | По умолчанию |
| --- | :-: |---------------------| --- |
| `currencyAmount` | Устанавливает количество монет, которое будет выброшено. | [Целое число](#integer) | нет |

<details>

<summary align="center"><b>Пример</b></summary>

<div align="left">

```yml
uniqueLootList:
  - currencyAmount: 344
    chance: 0.5
```
Это заставит моба выбросить 344 *Элитные Монеты* с 50% шансом выпадения.

</div>

</details>

### Выпадение ванильных предметов:

| Ключ | Подробности | Значения                | По умолчанию |
| --- | :-: |-----------------------| --- |
| `material` | Устанавливает материал выброшенного предмета. | [Материал](#material) | нет |

<details>

<summary align="center"><b>Пример</b></summary>

<div align="left">

```yml
uniqueLootList:
  - material: APPLE
    chance: 0.3
    amount: 5
```
Это заставит моба выбросить 5 *Яблок* с 30% шансом выпадения.

</div>

</details>

### Арены: В РАЗРАБОТКЕ (В настоящее время не работает)
При создании таблиц добычи для арен, пожалуйста, убедитесь, что ваша таблица добычи начинается с `rawArenaReward:` вместо `uniqueLootList:`.

| Ключ | Подробности | Значения              | По умолчанию |
| --- | :-: |---------------------| --- |
| `wave` | Устанавливает волну, на которой выпадет этот элемент добычи. Только для использования на аренах. | [Целое число](#integer) | нет |
| `level` | Устанавливает уровень выпадения добычи. Только для использования на аренах. | [Целое число](#integer) | нет |

<details>

<summary align="center"><b>Пример</b></summary>

<div align="left">

```yml
rawArenaReward:
  - material: BREAD
    wave: 1
    amount: 10
    chance: 0.5
  - filename: magmaguys_toothpick.yml
    wave: 1
    level: 2
```
Когда игроки пройдут первую волну, арена выбросит 10 *Хлеба* с 50% шансом выпадения и 1 *Зубочистку МагмаГая* уровня 2 со 100% шансом выпадения.

</div>

</details>

### Инстансовые Подземелья:

| Ключ | Подробности | Значения            | По умолчанию |
| --- | :-: |-------------------| --- |
| `difficultyID` | Устанавливает список сложностей инстансовых подземелий, для которых будет выпадать эта добыча. Только для использования в инстансовых подземельях. | [Строка](#string) | нет |

<details>

<summary align="center"><b>Пример</b></summary>

<div align="left">

```yml
uniqueLootList:
  - filename: magmaguys_toothpick.yml
    chance: 0.5
    difficultyID:
    - 1
    - 2
```
Это заставит моба выбросить 1 *Зубочистку МагмаГая* с 50% шансом выпадения, если игроки победили босса на сложности 1 или 2.

</div>

</details>
