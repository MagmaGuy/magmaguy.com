# Масштабирование урона и здоровья в EliteMobs

Этот документ представляет собой подробное описание того, как в EliteMobs происходит масштабирование урона и здоровья. Он должен использоваться как справочник при проектировании подземелий.

В нем основное внимание уделяется  Custom Bosses, но обычные элиты масштабируются таким же образом.

# Прежде чем начать!

Вам не нужно читать все это, если вы просто хотите внести быстрые изменения. Перейдите к разделу "Использование этих данных для подземелий", чтобы увидеть упрощенные быстрые расчеты и набор готовых рекомендуемых настроек!

# Масштабирование (в целом)

В EliteMobs по умолчанию боссы получают 7 единиц максимального здоровья и наносят на 1 единицу больше урона за уровень.

Оружие игроков наносит на 7 единиц больше урона за уровень, а броня снижает урон на 0,25 за уровень (но есть 4 слота брони, которые вносят вклад в общий показатель, что составляет 1 единицу). Наконец, игроки получают максимальное здоровье по мере разблокировки рангов престижа.

# Урон, наносимый  Custom Bosses

Формула урона для  Custom Bosses:

```
( ( ( базовый урон / 2 ) * множитель урона кастомного босса) + уровень босса - снижение брони игрока ) * множитель глобального урона
```

## базовый урон

Это может быть одна из двух вещей:

1.  Значение в файле mobproperties для типа существа босса.
2.  Поскольку это может быть сложно запомнить при проектировании подземелий и другого контента,  Custom Bosses  могут иметь поле normalizedCombat, которое применяет стандартный урон в 5 единиц, что имеет наилучшее масштабирование для прогрессии EliteMobs. В более поздней версии  normalizedCombat  будет включен по умолчанию для всех региональных боссов.

Это множитель урона в файле  Custom Boss.  По умолчанию равен 1.

## уровень босса

Как вы могли догадаться, это просто уровень босса. Это означает, что боссы наносят на 1 единицу больше урона за уровень.

## снижение брони игрока

Броня игрока снижает урон на 0,25 за уровень, и есть 4 слота брони, которые могут внести вклад в общий показатель. Кроме того, каждый уровень защиты от снарядов снижает урон от снарядов на 0,025.

## множитель глобального урона

Это множитель в MobCombatSettings. По умолчанию равен 1.

# Максимальное здоровье  Custom Bosses

Формула здоровья для  Custom Bosses:

```
( базовый максимальный уровень здоровья + уровень босса * 7 ) * множитель здоровья кастомного босса
```

## базовый максимальный уровень здоровья

Это может быть одна из двух вещей:

1.  Стандартное максимальное здоровье существа vanilla Minecraft.
2.  Поскольку это может быть сложно запомнить при проектировании подземелий и другого контента,  Custom Bosses  могут иметь поле normalizedCombat, которое применяет стандартное здоровье в 7 единиц, что имеет наилучшее масштабирование для прогрессии EliteMobs. Normalized Combat включен по умолчанию для всех региональных боссов.

# Урон, наносимый  Custom Bosses

Формула урона, которую игроки наносят  Custom Bosses, содержит слишком много мелких переменных, таких как критические удары и темпы ожидания, чтобы объяснить ее подробно здесь. Упрощенный вариант выглядит следующим образом:

```
урон = Уровень оружия
```

Это примерно так должен выглядеть урон в идеальном среднем сценарии.

# Использование этих данных для подземелий

**ПРИМЕЧАНИЕ: ЭТО ПРЕДПОЛАГАЕТ, ЧТО ВЫ ИСПОЛЬЗУЕТЕ ЗНАЧЕНИЯ `normalizedCombat`!** Эти значения были специально созданы для облегчения балансировки подземелий.

Часть сложности босса заключается в том, чтобы сделать его соответствующим по здоровью и урону для встречи. Однако имейте в виду, что это только часть, и способности также очень важны.

Ниже представлена правильная форма для файла  Custom Boss.

## Простые быстрые расчеты

1.  EliteMobs имеет 7 ударов как количество ударов, которое игрок с оружием того же уровня, что и босс, наносит, чтобы убить его, предполагая, что нет модификаторов.
    1.  Это означает, что множитель 2.0 означает, что игрокам нужно будет ударить босса 2 x 7 = 14 раз. 0.5 означает 7 / 2 = 3.5.
2.  Базовый урон normalizedCombat, предполагая, что броня игрока того же уровня, что и босс, составляет 2.5 единицы урона или 1.25 сердца урона за удар.
    1.  Это означает, что множитель 2.0 означает, что игроки будут терять по 3 сердца за удар, а множитель 0.5 заставит игроков терять по 0.75 сердца за удар.

**Помните, что игроки, как правило, отдают приоритет оружию и часто оказываются с гораздо худшей броней, чем оружием. Если вы не проектируете очень специфичные типы арен с дальним боем, вам, вероятно, не нужно наносить много урона за удар.**  Протестируйте свой контент!

## Создание обычного моба подземелья

С масштабированием  normalizedCombat  можно применить очень минималистичную конфигурацию:

```yaml
normalizedCombat: true
healthMultiplier: 1.0 #4 hits to slay 
damageMultiplier: 1.0 #1.5 hearts of dmg
```

**Примечание: так как 1.0 - это значение по умолчанию, это означает, что вы можете просто ничего не задавать**.

## Создание "пушечного мяса"

"Пушечное мясо" - это многочисленные, но не очень опасные мобы:

```yaml
normalizedCombat: true
healthMultiplier: 0.7 #3 hits to slay 
damageMultiplier: 0.5 #0.5 hearts of dmg
```

## Создание подкреплений

Подкрепления должны легко умирать, но представлять опасность с точки зрения урона (эти значения рекомендуются для 4+ мобов ближнего боя, фактическое использование может варьироваться):

```yaml
normalizedCombat: true
healthMultiplier: 0.25 #1 hit to slay 
damageMultiplier: 0.6 #1 heart of dmg
```

## Создание мини-боссов

Мини-боссы должны удерживать позицию и представлять механический вызов для игроков. Это должен быть тест на мастерство, который длится какое-то время, но не что-то слишком смертельное:

```yaml
normalizedCombat: true
healthMultiplier: 3.0 #10 hits to slay 
damageMultiplier: 1.2 #2 hearts of dmg
```

## Создание боссов

Боссы - это настоящий вызов, кульминация подготовки в подземелье и настоящий тест на мастерство, когда на кону все. Встречи должны быть долгими, и смерть должна быть реальной угрозой.

```yaml
normalizedCombat: true
healthMultiplier: 7.0 #23 hits to slay 
damageMultiplier: 1.4 #2.5 hearts of dmg
```

## Создание Мировых боссов

Мировые боссы опасны, но самое главное - у них много здоровья, и они вызывают много подкреплений/имеют много AoE-атак. Они предназначены для борьбы с большими группами игроков в течение максимально долгого времени, которое должна длиться встреча с элитами. Они не очень смертельны, но у них много механик, которые могут одолеть неподготовленных игроков.

```yaml
normalizedCombat: true
healthMultiplier: 40.0 #160 hits to slay 
damageMultiplier: 1.5 #2.5-3 hearts of dmg
```

Это лишь приблизительные рекомендуемые значения, и любые окончательные значения должны быть настроены в соответствии с самими встречами.
