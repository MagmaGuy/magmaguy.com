# События скриптов элиты

Скрипты элиты основаны на событиях. Это означает, что они запускаются при срабатывании определенного события. Ниже приведены текущие действительные события:

| Событие                       | Подробности                                   | Прямая цель [[?]($language$/elitemobs/elitescript_targets.md)] |
| :---------------------------- | :-------------------------------------------- | :-------------------------------------------------------------: |
| EliteMobDamagedByEliteMobEvent | Элита повреждена другой элитой                |                                ❌                                 |
| EliteMobDamagedByPlayerEvent | Элита повреждена игроком                      |                         Повреждающий игрок                          |
| EliteMobDamagedEvent          | Элита повреждена чем угодно                   |                                ❌                                 |
| EliteMobDeathEvent            | Смерть элиты                                  |                                ❌                                 |
| EliteMobEnterCombatEvent      | Элита вступает в бой с игроком                |                         Вступающий в бой игрок                          |
| EliteMobExitCombatEvent       | Элита выходит из боя                          |                                ❌                                 |
| EliteMobHealEvent             | Полное исцеление элиты                        |                                ❌                                 |
| EliteMobSpawnEvent            | Появление элиты                               |                                ❌                                 |
| EliteMobTargetPlayerEvent     | Элита выбирает игрока целью                   |                         Выбранный целью игрок                          |
| PlayerDamagedByEliteMobEvent  | Элита наносит урон игроку                     |                          Поврежденный игрок                          |
| ElitePhaseSwitchEvent         | Появление босса при смене фазы                |                                ❌                                 |
| ZoneEnterEvent                | Сущность входит в зону. Требуется установить зону! | Сущность, вошедшая в зону                                 |
| ZoneLeaveEvent                | Сущность покидает зону. Требуется установить зону! | Сущность, покинувшая зону                                 |

**Примечание: "Элита" относится к любой агрессивной сущности, созданной плагином, независимо от того, является ли она подкреплением, боссом или чем-либо еще.**
**Примечание: ZoneEnterEvent и ZoneLeaveEvent требуют значительных вычислительных ресурсов, поэтому используйте их экономно!**

`Прямая цель` используется Целями скриптов, подробнее об этом [здесь]($language$/elitemobs/elitescript_targets.md).

`ZoneEnterEvent` и `ZoneLeaveEvent` — это особые события, которые позволяют скриптерам запускать действия на основе входа и выхода сущностей из зон, установленных в скриптах. DIRECT_TARGET для ZoneEnterEvent и ZoneLeaveEvent — это сущность, которая вошла или вышла (это означает, что вы не можете использовать прямую цель как цель зоны), и из-за особенностей внутренней логики для сканирования входа и выхода могут использоваться только абсолютные точки (конкретные координаты) и точки, относительные к боссу (нельзя использовать зоны, нацеленные на игроков).

В каждом скрипте может быть только одна секция [Event]($language$/elitemobs/elitescript_events.md). Однако в качестве триггеров для скрипта может быть установлено несколько событий:

<div align="center">

<details>

<summary><b>Пример</b></summary>

<div align="left">

```yaml
eliteScript:
  Example:
    Events:
    - EliteMobDamagedByPlayerEvent
    - EliteMobTargetPlayerEvent
    Actions:
    - action: PUSH
      Target:
        targetType: SELF
      vValue: 0,.3,0
    Cooldowns:
      local: 60
      global: 20
```

Этот скрипт заставит элиту подпрыгнуть при ударе игроком или при выборе игрока целью.

</details>
