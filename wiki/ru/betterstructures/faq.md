# Проверка, где появляются структуры

- Если вы являетесь администратором сервера и не отключили предупреждения, вы будете уведомлены в чате, когда появится структура.
- Нажмите на сообщение в чате, чтобы телепортироваться к структуре (не работает в Bedrock edition).
- Включить/отключить предупреждения с помощью `/betterstructures warn`.

# Почему структуры могут не появляться

- Обычно структуры не появляются в уже исследованных областях. Посетите новые области для поиска структур.

# EliteMobs и BetterStructures

## Что предлагает EliteMobs
- Узнайте больше об EliteMobs: [Wiki EliteMobs](#)
- Он добавляет боссов в структуры.

## Получение готового контента EliteMobs
- Доступно на [magmaguy.itch.io](https://magmaguy.itch.io/) и [patreon.com/magmaguy](https://www.patreon.com/magmaguy)

## Использование EliteMobs без других функций
- Вы можете отключить другие функции EliteMobs. Чтобы отключить, измените настройки в:
    - events.yml: `timedEventsEnabled: false` и `actionEventsEnabled: false` отключают все события
    - MobCombatSettings.yml: `doNaturalMobSpawning: false` отключает естественную генерацию элитных мобов

## Добавление босса EliteMobs в вашу структуру

- Добавьте `[elitemobs]` в первую строку вашей таблички
- Добавьте имя файла босса во вторую/третью/четвертую строки таблички, например, `test_boss.yml`
- Дополнительная информация [здесь]($language$/betterstructures/creating_structures.md)

# BetterStructures с другими плагинами

## MythicMobs
- Генерация мобов MythicMobs с помощью табличек с `[mythicmobs]`, за которым следует идентификатор моба в других строках таблички.
- EliteMobs лучше подходит для боссов с BetterStructures, потому что я сделал оба.

## MMOItems
- Добавьте MMOItems с помощью `mmoitems=<TYPE>@<ITEM-ID>` в конфигурационный файл treasure.

## Пользовательские предметы из других плагинов
- Используйте команду lootify для добавления любых предметов в сундуки. Смотрите [страницу команд]($language$/betterstructures/commands.md).

# Влияние на производительность и его минимизация
- Генерация большого количества структур может замедлить ваш сервер.
- Используйте [Chunky](https://www.spigotmc.org/resources/chunky.81534/) для предварительной генерации вашего мира и избегания проблем с производительностью.
- Если вы не используете EliteMobs, вы можете удалить BetterStructures после генерации мира.
- С EliteMobs, оставляйте BetterStructures для управления защищенными зонами, что оказывает минимальное влияние на производительность.
