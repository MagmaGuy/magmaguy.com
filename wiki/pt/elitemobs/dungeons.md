# Tipos de Dungeon

***Observação: o seguinte descreve o conteúdo pré-fabricado do EliteMobs. O que você faz no seu servidor pode ser personalizado, e os recursos descritos abaixo são simplesmente uma maneira conveniente de agrupar diferentes tipos de conteúdo.*** O conteúdo que você cria para o seu servidor pode seguir formatos e regras totalmente diferentes.

## Dungeons Abertas

As dungeons abertas estão sempre abertas para todos os jogadores. Isso significa que qualquer quantidade de jogadores pode entrar na dungeon a qualquer momento para lutar contra os mesmos chefes. Isso é ótimo para se agrupar casualmente e farmar, mas pode ser complicado devido aos temporizadores de respawn.

Para os fãs de WoW / FF14, este é o equivalente a zonas de mundo aberto com chefes mundiais.

### O que é um Covil?
Covis são um tipo de dungeon pequena que se concentra em uma grande luta de chefe.

A estrutura: Os covis geralmente são restritos a uma construção em torno de uma grande arena de combate e são a menor categoria de tamanho do EliteMobs.

As lutas: Espera-se que os jogadores se agrupem em grupos de 5 para desafiar o chefe, pois geralmente são muito desafiadores.

Para os fãs de FF14, este é o equivalente a chefes mundiais.

### O que é uma Mini-Dungeon?
Mini-dungeons são um tipo de dungeon de tamanho médio que geralmente contém 3 a 15 mini-chefes e um grande chefe, bem como muitos mobs "lixo" que devem ser "farmados" por moedas e equipamentos pelos jogadores.

A estrutura: As menores mini-dungeons têm o tamanho de grandes navios, e a maior mini-dungeon pode ser um mapa bastante expansivo, com centenas de mobs vagando para lutas.

As lutas: Mini-dungeons podem ser jogadas sozinhas por jogadores veteranos, mas grupos de 2 a 3 jogadores são recomendados.

Para os fãs de WoW / FF14, este é o equivalente a chefes mundiais em zonas que são desafiadoras, mas recompensadoras para farmar.

### O que é uma Aventura?
Aventuras são mapas de aventura completos. Eles incluem linhas de missões, NPCs, grandes quantidades de mini-chefes e chefes e podem até conter suas próprias arenas ou dungeons dentro deles.

A estrutura: Estes são sempre mapas muito grandes, com dezenas de grandes construções divididas em diferentes zonas. Eles também podem incluir mundos secundários e terciários por vários motivos.

As lutas: Aventuras podem ser jogadas sozinhas por jogadores veteranos, mas é recomendável agrupar outro jogador.

Para os fãs de WoW / FF14, este é essencialmente todo o jogo base de missões, encontrar dungeons e lutar contra chefes ao longo de uma linha de missões.

## Dungeons Instanciadas

As dungeons instanciadas são dungeons que criam mundos conforme a necessidade, permitindo que o servidor execute tantas lutas simultâneas para quantos jogadores forem necessários. É a melhor opção para criar conteúdo difícil, mas justo, e é o tipo de conteúdo menos explorável.

Existem muitos recursos para Dungeons Instanciadas, sobre os quais você pode ler [aqui]($language$/elitemobs/instanced_dungeon_difficulty.md&section=how-it-works).

Para os fãs de WoW e FF14, este é o equivalente ao instanciamento de dungeons / obrigações.

### Santuários
Santuários são o equivalente a Covis instanciados. Isso significa que eles se concentram em torno de uma grande luta de chefe. Este é o tipo mais difícil de conteúdo EliteMobs, especialmente para as dificuldades mais altas.

A estrutura: Geralmente são mapas pequenos, mas podem ter arenas de combate muito dinâmicas. Eles geralmente se concentram em uma única sala grande.

As lutas: Destinadas a serem lutadas por 5 jogadores. Nas dificuldades mais altas, este é o conteúdo EliteMobs mais difícil que existe.

Para os fãs de FF14, este é o equivalente aos Trials.

### Dungeons
Dungeons são o equivalente a Mini-dungeons instanciadas. Este é o tipo mais tradicional de dungeon de MMORPG, onde os jogadores se agrupam e passam por uma série de mobs lixo e mini-chefes para lutar contra um chefe final.

Para os fãs de WoW e FF14, este é o equivalente às Dungeons.

A estrutura: Geralmente são mapas de tamanho médio, maiores do que as mini-dungeons, mas muito menores do que as Aventuras.

As lutas: Destinadas a serem lutadas por grupos de 5, mais difíceis do que as dungeons abertas, mas mais fáceis do que os Santuários.

### Arenas

Arenas são áreas de sobrevivência baseadas em ondas, nas quais os jogadores são recompensados ​​por derrotar ondas. As arenas geralmente têm alguma história em torno delas.

A estrutura: Do tamanho de uma arena e com formato de arena.

As lutas: Podem ser jogadas sozinhas, mas podem ser desafiadas por até 5 jogadores.
