# Porquê?

Honestamente, nunca gostei de como a Mojang implementou o sistema alimentar. Constantemente tira os jogadores do que estão fazendo para preencher uma barra sem sentido.
<br>Eu entendo que a comida é importante para um jogo de sobrevivência, mas acho que manter pressionado o botão direito do mouse por alguns segundos a cada dois minutos não torna uma mecânica de jogo convincente.





E quanto a outras comidas?
É melhor deixar a gestão dos alimentos crus e dos alimentos que fornecem buffs por conta do jogador. Não quero que os jogadores desperdiçem alimentos crus que eles estão colhendo para cozinhar, nem estou a criar este plugin para tornar as pessoas mais preguiçosas em relação à comida.

Se as pessoas estiverem dispostas a se esforçar para cozinhar ou fabricar sua comida, elas poderão reabastecer automaticamente. Parece um bom equilíbrio, certo?

# Permissões/Comandos/Configurações:

- `betterfood.user` - O plugin só se aplica aos jogadores com esta permissão.
- `/betterfood toggleEating` - Ativa/desativa a cura automática para o jogador.
- `/betterfood toggleMessage` - Ativa/desativa as mensagens de alimentação para o jogador.
